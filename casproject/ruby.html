Setting up the Environment

Welcome! As the Flatiron Schools describes it, "Ruby is a simple, yet powerful programming language that you can use for everything from web development to processing text to building games." Some sites you may know of that are coded with Ruby are: SoundCloud, AirBnB, and Groupon.

This Ruby course is crafted from a selection of handpicked courses, so that you can get the most out of your lessons. To start you off, there are a few accounts you will have to sign up for. First and foremost, you will be using Code Academy. Head on over there and sign up.
Lesson 1: The Basics

In this lesson you will be learning about the fundamentals of Ruby, and the building blocks Ruby programmers use and manipulate, like strings, and floats.

To start your first lesson, click here. Continue until you are prompted with the "First Lesson" completion badge.

A string can be any sort of data. It can hold words, numbers, or any other character. Digits, or floats, are numbers, that can be mathematically manipulated and sorted.
Lesson 2: Variables and Methods

For lesson 2, you will be taking what you have learned in the first lesson and applying them to what we call methods. To start your lesson,click here.

"A method is a set of instructions that can be reused in a program. Ruby comes with a set of predefined methods, but you can also write your own methods. A commony used method, especially when you're first learning Ruby, is puts.For instance, puts 'hello world' will print 'hello world' to the screen."
Lesson 3: Hashes and Arrays

For lesson 4, you will start building on the data you can store and use in Ruby. These are called hashes and arrays.To begin, click here.

For more information on arrays and hashes, I recommend Ruby in 100 Minutes - Arrays and Hashes.
Lesson 4: Conditionals

Lesson 4 will teach you want an if statement is and how it is used. By the end of the lesson, you should be able to implement and if statement with over 3 branches, which is also called refactoring, or conditionals. To have some fun click here!

"These 'if' experssions essentially check whether a condition is true or not. In Ruby they are created by writing the if keyword, followed by a condition, optionally followed by the then, keyword, then the code you want to execute if the condition is true, and then finally the end keyword to end the structure." To read more, take a look at this study guide from DevHQ.
Lesson 5: Object Orientation

For this final module in your intro to Ruby, you will learn about object orientation.By completion, you will know how to create a class with data, or attributes, and write instances of that class. Have at it.

You can think of a class as a template for certain information to be put into. For example, each user on twitter has a "class" of their personal information, which are the attributes, that you provide when signing up, this includes, name, birthday, username, and password. For more information on object orientation, check out this resource.
